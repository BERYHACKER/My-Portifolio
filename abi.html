ABI Calculator

Interactive Web Application for Advanced Calculations

Project Overview

The ABI Calculator is an interactive web application designed to perform complex calculations with a focus on user experience. Built entirely with vanilla JavaScript, HTML5, and CSS3, it demonstrates clean code principles, responsive design, and intuitive UI/UX patterns.

Design & User Experience

The calculator was designed with user experience as the primary focus. The interface uses clear typography, intuitive button layout, and immediate visual feedback. The color scheme utilizes the modern cyan (#00f7ff) and green (#78e693) gradient to match the portfolio theme.

JavaScript Implementation

The calculation engine is built using vanilla JavaScript with proper separation of concerns. Event listeners handle user input, a calculation module manages operations, and DOM manipulation updates the display in real-time.

IoT LED Control System

Remote LED control using ESP32, Wi-Fi, and MQTT Protocol

Project Overview

The IoT LED Control System is a comprehensive solution for remotely controlling LED lights using modern IoT technologies. This project demonstrates integration of embedded systems, wireless communication, and web interfaces to create a scalable IoT application that allows users to control multiple LEDs remotely.

Hardware Setup

The hardware consists of an ESP32 microcontroller as the central control unit, connected to relay modules that switch LED power on/off. Additional sensors monitor LED status and power consumption. System powered by 5V supply.
